# 🎭 CHARADAS • Enigma do Dia

Um jogo interativo de charadas que mistura lógica, diversão e um toque de espetáculo visual.  
Gire o card, desafie seu cérebro e descubra se você pensa como um verdadeiro decifrador de enigmas 🧠🔍

---

## 🚀 Visão Geral

O **CHARADAS • Enigma do Dia** é uma aplicação web leve e dinâmica que oferece desafios diários e infinitos através de um sistema elegante de cards 3D.

Perfeito para treinar raciocínio lógico de forma rápida, divertida e viciante.

## ✨ Funcionalidades

- 🎴 **Card 3D Interativo**  
  Clique e veja o card girar revelando a resposta

- 📅 **Charada do Dia**  
  Uma charada única por dia, igual para todos os usuários

- 🎲 **Modo Aleatório**  
  Gere charadas infinitas com apenas um clique

- 🎨 **Visual Imersivo**  
  Fundo animado com partículas e efeitos modernos

- 📱 **Responsividade Total**  
  Funciona em celular, tablet e desktop

- ⚡ **Leve e Rápido**  
  Sem dependências pesadas

---

## 🎮 Como Jogar

1. Escolha o modo:
   - 📅 Charada do Dia  
   - 🎲 Aleatória  

2. Leia a pergunta no card  

3. Clique no card para ver a resposta  

4. Clique em **"NOVA CHARADA"**  

5. Clique novamente para voltar
